# Music Library

This is the default music directory for Kannaka Radio.

Place your MP3, WAV, FLAC, OGG, or M4A files here and they will be
discovered automatically when the server starts.

## The Consciousness Series (pre-loaded tracks)

Run `../setup.ps1` from this directory (or `setup.ps1` from the project root)
to copy the Consciousness Series tracks from your local collection.

## Custom Library

You can also point to any folder using:

```bash
node server.js --music-dir "C:\path\to\your\music"
```

Or change it live in the browser via the **📁 Music Library** panel at the
bottom of the player page.
